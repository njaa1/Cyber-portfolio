# Caesar Cipher

Simple encryption/decryption tool using Caesar cipher.

## What is Caesar cipher?

Each letter gets shifted by a number of positions in the alphabet.

For example with shift of 3:
- A becomes D
- B becomes E
- Z becomes C

## How to use

```bash
python3 crypto.py
```

Then choose 1 to encrypt or 2 to decrypt, enter your message and the shift number.
